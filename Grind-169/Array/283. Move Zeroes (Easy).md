283. Move Zeroes (Easy)
===

Problem: https://leetcode.com/problems/move-zeroes/

---

1. two pointers
```python
# time: O(n), space: O(1)
class Solution:
    def moveZeroes(self, nums: List[int]) -> None:
        i = 0
        for j in range(len(nums)):
            if nums[i] == 0 and nums[j] != 0:
                nums[i], nums[j] = nums[j], nums[i] # 數字對調
            if nums[i] != 0:
                i += 1

        return nums   
```
