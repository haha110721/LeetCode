Maximum Depth of Binary Tree (Easy)
===

Problem: https://leetcode.com/problems/maximum-depth-of-binary-tree/

---

1. (我要想一下) DFS 遞迴 
```python
# time: O(n), space: O(h)

# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def maxDepth(self, root: Optional[TreeNode]) -> int:
        
        def dfs(root, depth):  # dfs(root, depth) 是一個傳入目前節點和目前深度的遞迴函式
            if not root:  # 如果走到 None（空節點），就代表走到底了，回傳當前的 depth
                return depth
            return max(dfs(root.left, depth + 1), dfs(root.right, depth + 1))  # 每往下一層，depth 就 +1
                       
        return dfs(root, 0)
```

```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def maxDepth(self, root: Optional[TreeNode]) -> int:
        if not root:
            return 0  # 空節點，深度為 0

        # 遞迴找左右子樹的最大深度
        left_depth = self.maxDepth(root.left)
        right_depth = self.maxDepth(root.right)

        return max(left_depth, right_depth) + 1  # 加上目前這一層
```
