Longest Palindrome (Easy)
===

Problem: https://leetcode.com/problems/longest-palindrome/

---

1. hash map
```python
class Solution:
    def longestPalindrome(self, s: str) -> int:
        res = 0
        
        check_dict = {}
        for i in s:
            if i in check_dict:
                del check_dict[i]
                res += 2
            else:
                check_dict[i] = 1
        
        if check_dict:
            return res + 1
        else:
            return res
```

2. 用 set
```python
# time: O(n), space: O(n)

class Solution:
    def longestPalindrome(self, s: str) -> int:
        ss = set()

        for letter in s:
            if letter not in ss:
                ss.add(letter)
            else:
                ss.remove(letter)
        
        if len(ss) != 0:
            return len(s) - len(ss) + 1 
        else:
            return len(s)  # 表示 s 的所有字母都有用到
```

3. (別人寫的)
```python
class Solution:
    def longestPalindrome(self, s: str) -> int:
        odd_count = 0
        d = {}
        for ch in s:
            if ch in d:
                d[ch] += 1
            else:
                d[ch] = 1

            if d[ch] % 2 == 1:  # 檢查是不是奇數
                odd_count += 1
            else:
                odd_count -= 1

        if odd_count > 1:
            return len(s) - odd_count + 1
        return len(s)
```
